================================================================================
  TROUBLESHOOTING QUICK REFERENCE
================================================================================

Problem: Service won't start
Solution: journalctl -u cv-analyzer -n 50

Problem: Permission denied
Solution: chmod 755 ~/public_html/cv-analyzer-service

Problem: Port 8001 already in use
Solution: lsof -i :8001 then kill -9 <PID>

Problem: CORS errors
Solution: Update ALLOWED_ORIGINS in .env

Problem: Spacy model missing
Solution: python -m spacy download en_core_web_sm

Problem: Dependencies not found
Solution: ./deploy.sh update-deps
